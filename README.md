# 🎬 Netflix Clone

A pixel-perfect frontend clone of the Netflix landing page, built using pure **HTML** and **CSS**. This project was created to practice semantic HTML, modern CSS layouts (Flexbox/Grid), and responsive design.

---

## 🚀 Features

* **Responsive Design:** Fully optimized for desktop, tablet, and mobile screens.
* **Accordion FAQ:** Interactive Frequently Asked Questions section (styled with CSS).
* **Netflix Styling:** Authentic color palette, typography, and button hover states.
* **Custom Assets:** High-quality background images and icon integration.

---

## 🛠️ Tech Stack

* **HTML5:** For semantic web markup.
* **CSS3:** For custom styling, layouts, and animations.
